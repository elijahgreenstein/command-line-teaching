.. _cli-commands-clt-addmod:

``clt addmod``
============================================================

Summary
------------------------------------------------------------

Set up a new module in a directory.

Usage
------------------------------------------------------------

.. code-block:: console

  $ clt addmod <directory>

Examples
------------------------------------------------------------

.. code-block:: console

  $ clt addmod Week-01

Details
------------------------------------------------------------

CLT prompts the user for basic module information: title, position among other modules, prefix, and default quiz date.

CLT then creates the named directory, a settings file, and several default files for learning materials (introduction page, quiz, and discussion board).
