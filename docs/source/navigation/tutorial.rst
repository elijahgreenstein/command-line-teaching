Tutorial
============================================================

In this tutorial, we will use the Command Line Teaching (CLT) toolkit to set up a new course, create a new module, and upload module contents to Canvas LMS.

Prerequisites
------------------------------------------------------------

* An installation of `Python <python_>`_. Install this software first.
* Familiarity with the `CommonMark <commonmark_>`_ specification of Markdown syntax.
